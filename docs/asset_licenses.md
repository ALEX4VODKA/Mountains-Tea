# 素材授权记录

本文件用于记录项目中使用的所有外部素材来源和许可证信息。

## 基本原则

1. 不使用授权不清楚的素材。
2. 免费素材也要记录来源。
3. 需要署名的素材必须保留作者信息。
4. 购买素材也要记录购买来源和授权范围。
5. 参考图和正式素材分开管理。
6. 不把其他商业游戏的截图或拆包素材放进正式项目。

## 推荐优先级

优先使用：

1. 自制素材
2. CC0 素材
3. 明确允许免费使用的素材
4. 已购买且授权清楚的素材

谨慎使用：

1. 需要署名的素材
2. 需要相同协议分享的素材
3. 授权说明不够清楚的素材

不要使用：

1. 其他游戏拆包素材
2. 授权不明的网图
3. 只在社交平台看到但无授权来源的图
4. 不清楚能否商用或再发布的 AI 素材包

# 素材授权记录

本文件用于记录项目中使用的所有外部素材来源和许可证信息。

## 基本原则

1. 不使用授权不清楚的素材。
2. 免费素材也要记录来源。
3. 需要署名的素材必须保留作者信息。
4. 购买素材也要记录购买来源和授权范围。
5. 参考图和正式素材分开管理。
6. 不把其他商业游戏的截图或拆包素材放进正式项目。

## 推荐优先级

优先使用：

1. 自制素材
2. CC0 素材
3. 明确允许免费使用的素材
4. 已购买且授权清楚的素材

谨慎使用：

1. 需要署名的素材
2. 需要相同协议分享的素材
3. 授权说明不够清楚的素材

不要使用：

1. 其他游戏拆包素材
2. 授权不明的网图
3. 只在社交平台看到但无授权来源的图
4. 不清楚能否商用或再发布的 AI 素材包

## 第一阶段最少下载清单

第一阶段正式下载时，优先只下载以下资源，避免素材过多导致风格混乱：

1. 1 个主角 4 方向行走 spritesheet
2. 1 套自然地图 tileset
3. 1 套木屋或村庄建筑 tileset
4. 1 套茶铺室内或 cozy shop interior tileset
5. 1 套物品图标，至少覆盖茶叶、水、木柴、茶、金币
6. 1 套简单像素 UI 包
7. 1 个可用于游戏内嵌的像素字体
8. 1 段可循环白天环境音乐
9. 1 个基础音效包，至少覆盖采集、制作、售卖或金币

## 下载前授权核对清单

每个候选素材下载前都检查：

- 是否有明确作者或发布者
- 是否有明确许可证名称或商店授权说明
- 是否允许用于游戏项目
- 是否允许修改、裁切、调色或重新打包进 Godot 项目
- 是否要求署名
- 是否限制商用
- 是否要求相同协议分享
- 是否有单独的字体、音乐或音效授权条款
- 是否需要保留原始 license 文件

下载后要做：

- 在“正式使用素材”中新增一条记录
- 保存原始 URL
- 保存许可证文本或截图到项目外的素材管理位置
- 如果需要署名，在发布说明或 credits 中保留作者信息

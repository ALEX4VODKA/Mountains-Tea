# 山间茶铺第一张地图最小可玩布局

## 地图目标

第一张地图应服务 5 到 10 分钟 demo：玩家从茶铺醒来，出门采集三种资源，回到茶铺制作 `simple_tea`，卖给访客，然后睡觉进入下一天。

地图保持小而清楚，适合 2D 俯视角像素风。建议整体尺寸约为 40 x 28 格 tile，玩家从中心茶铺出发，所有关键点都在 10 到 20 秒步行范围内。

## 整体布局

布局思路：茶铺放在地图中心偏下，门前小路作为主导向。左上是树林采茶区，右上是溪流取水点，左下靠近茶铺放柴堆，茶铺室内或后侧放睡觉点。玩家不需要大地图探索，只需要围绕茶铺做一圈短路线。

```text
+----------------------------------------+
|  树林采茶区                             |
|  T T T T        山石/草地        溪流取水点|
|  T 茶叶 T                         ~ ~ ~ |
|  T T T T            小桥/石头     ~ 水 ~ |
|       \              路径         ~ ~ ~ |
|        \                          /     |
|         \                        /      |
|          =========门前小路========       |
|                    |                   |
|              +----------+              |
|              |  茶  铺  |              |
|              | 制茶/售卖 |              |
|              | 睡觉点 B |              |
|              +----------+              |
|                 |                      |
|       柴堆/木柴采集点                  |
|       W W W                             |
|       W 木柴 W        空地/访客停留点    |
|       W W W                             |
+----------------------------------------+

图例：
T = 茶树/茶叶采集点
~ = 溪流
W = 柴堆/木柴采集点
B = 睡觉点
=== = 门前小路
```

## 区域说明

### 1. 茶铺

茶铺是玩家的核心基地，位于地图中心偏下，方便玩家从任意采集点快速返回。

玩法作用：

- 玩家醒来的起点。
- 放置制茶互动点，用于把 `tea_leaves + spring_water` 制作为 `simple_tea`。
- 放置售卖互动点，用于出售 `simple_tea` 并增加金钱。
- 提供安全、温暖、可识别的视觉中心。

建议表现：

- 木质小屋、暖色窗光、门口灯笼或小灯。
- 门口留出 3 到 5 格空地，方便玩家移动和与访客互动。

### 2. 门前小路

门前小路从茶铺门口向上延伸，再分叉到树林和溪流。

玩法作用：

- 引导玩家自然理解地图结构。
- 连接茶铺、采茶区、取水点和木柴点。
- 让玩家不依赖复杂指引，也能找到资源。

建议表现：

- 使用浅色石板、泥土路或碎石路。
- 路线短、分叉少，避免第一张地图显得复杂。

### 3. 树林采茶区

树林采茶区位于地图左上角，是 `tea_leaves` 的来源。

玩法作用：

- 玩家靠近茶树或茶叶闪光点后采集 `tea_leaves`。
- 可以放 3 到 5 个采集点，足够完成第一次制茶。
- 让玩家体验出门采集和返回茶铺的基本循环。

建议表现：

- 茶树、普通树、草丛混合摆放。
- 采集点不要藏太深，保持第一眼可读。
- 可用一小片雾或树影强化山林氛围。

### 4. 溪流取水点

溪流取水点位于地图右上角，是 `spring_water` 的来源。

玩法作用：

- 玩家到溪流边互动获得 `spring_water`。
- 与茶叶采集点分布在不同方向，形成一条很短的探索路线。
- 可以作为地图的自然边界，避免玩家走出太远。

建议表现：

- 溪流横向或斜向穿过右上角。
- 在靠近小路的位置放一个明确的取水点，例如石阶、木桶或水边亮点。
- 溪流旁可放石头、小桥或水草，提升场景层次。

### 5. 柴堆或木柴采集点

柴堆位于茶铺左下或门口附近，是 `firewood` 的来源。

玩法作用：

- 玩家采集 `firewood`，为后续制茶、炉火或日常任务预留空间。
- 第一个 demo 中即使配方暂时不消耗木柴，也能作为第三种资源教学。
- 因为木柴靠近茶铺，玩家可以快速完成一次采集，不拖慢节奏。

建议表现：

- 木柴堆、砍好的原木、小斧头或树桩。
- 放在茶铺旁边比放在远处更适合第一张小地图。

### 6. 睡觉点

睡觉点放在茶铺室内，或茶铺后侧一个明确的小床/蒲团位置。

玩法作用：

- 玩家互动后触发 day/night transition，并进入下一天。
- 作为一天流程的结束按钮。
- 帮助 demo 形成完整闭环：采集、制作、售卖、睡觉。

建议表现：

- 小床、草席、被褥或屏风旁的休息区。
- 睡觉点应与制茶/售卖点保持一点距离，避免误触。

## 推荐玩家路线

```text
醒来
  |
  v
出茶铺
  |
  v
去左上树林采集 tea_leaves
  |
  v
去右上溪流采集 spring_water
  |
  v
回茶铺制作 simple_tea
  |
  v
在茶铺门口或柜台出售 simple_tea
  |
  v
采一点 firewood 作为资源教学
  |
  v
回睡觉点进入下一天
```

## 尺寸和节奏建议

- 从茶铺门口到树林采茶区：约 8 到 12 秒步行。
- 从树林到溪流取水点：约 8 到 12 秒步行。
- 从溪流回茶铺：约 10 到 15 秒步行。
- 柴堆离茶铺最近，约 3 到 6 秒步行。
- 第一天完整流程不超过 3 分钟，剩余时间用于熟悉移动、观察场景和重复售卖。

## 最小可玩检查表

- 玩家出生点在茶铺内或茶铺门口。
- 茶铺门口能清楚看见小路。
- 玩家沿小路能找到树林采茶区。
- 玩家沿小路能找到溪流取水点。
- 柴堆靠近茶铺，容易发现。
- 制茶点、售卖点、睡觉点互相分开，避免混淆。
- 地图边界用树林、山石、溪流或栅栏自然封住。
- 所有关键互动点都在一屏到两屏范围内。

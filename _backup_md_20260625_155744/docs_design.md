# 游戏设计文档：山间茶铺

## 1. 一句话概念

《山间茶铺》是一款 2D 俯视角像素风休闲探索游戏。玩家经营一间山间小茶铺，在附近的森林、溪流和山路采集资源，回到茶铺制作简单茶饮和小物品，卖给路过的旅人，并在昼夜循环中感受安静、温暖、风景优美的小世界。

## 2. 项目目标

这是一个适合 Godot 初学者完成的小体量项目。

第一目标不是做完整商业游戏，而是完成一个 5 到 10 分钟可玩的 demo。

## 3. 核心体验

玩家应该感受到：

- 安静
- 温暖
- 治愈
- 有一点探索欲
- 在小世界里慢慢做事的安心感

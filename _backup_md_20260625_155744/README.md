# 山间茶铺 / Mountain Tea House

## 项目简介

《山间茶铺》是一款使用 Godot 4 制作的 2D 俯视角像素风休闲探索游戏。

玩家经营一间位于山间的小茶铺，白天在附近的树林、溪流和山路采集资源，回到茶铺制作简单茶饮或小物品，并卖给路过的旅人。游戏强调安静、温暖、风景优美和低压力的探索体验。

## 当前定位

- 类型：2D 俯视角像素休闲探索
- 引擎：Godot 4
- 渲染器：Compatibility
- 语言：GDScript
- 美术方向：像素风、山间、森林、溪流、暖光、雾气、轻 2.5D
- 开发方式：Codex 每次只完成一个小系统

# 山间茶铺 / Mountain Tea House

## 项目简介

《山间茶铺》是一款使用 Godot 4 制作的 2D 俯视角像素风休闲探索游戏。

玩家经营一间位于山间的小茶铺。白天，玩家可以在茶铺周围的树林、山路和溪边采集资源；回到茶铺后，可以制作简单茶饮或小物品，并卖给路过的旅人。游戏重点不是战斗和复杂剧情，而是风景、氛围、探索、制作和慢节奏经营。

## 当前方向

- 引擎：Godot 4
- 渲染器：Compatibility
- 语言：GDScript
- 类型：2D 俯视角像素休闲探索
- 画风：像素风、自然风景、暖光、雾气、轻 2.5D 氛围
- 核心玩法：采集、背包、制作、售卖、昼夜循环、小屋生活
- 开发方式：使用 Codex 辅助开发，每次只实现一个小系统

## 核心体验

希望玩家感受到：

- 安静
- 温暖
- 治愈
- 有一点探索欲
- 有一个可以慢慢经营的小世界

## 核心玩法循环

醒来  
→ 出门探索  
→ 采集资源  
→ 回茶铺制作  
→ 售卖物品  
→ 天色变暗  
→ 睡觉进入下一天

## 第一阶段目标

完成一个 5 到 10 分钟可玩的 demo。

第一阶段只做最小闭环：

1. 玩家可以移动
2. 摄像机跟随玩家
3. 有一张小地图
4. 有茶铺、树林、溪流或取水点
5. 可以采集 3 种资源
6. 可以查看简单背包
7. 可以制作 simple_tea
8. 可以卖出 simple_tea 获得金币
9. 有基础昼夜变化
10. 可以睡觉进入下一天

## 第一阶段不做

为了避免项目失控，第一阶段暂时不做：

- 战斗
- 大型剧情
- 多地图开放世界
- 复杂 NPC 好感度
- 技能树
- 装备系统
- 随机地牢
- 多人联机
- 复杂经济系统
- 复杂存档系统

## 项目目录

- scenes：Godot 场景文件
- scripts：GDScript 脚本
- assets：美术、音频、字体、UI 等资源
- docs：项目设计、任务规划、美术方向和 Codex 工作流
- builds：导出的游戏版本

## 开发原则

这个项目的目标不是一次做出很大的游戏，而是完成一个小而完整的作品。

每次只做一个小系统。  
每个系统都要能在 Godot 中单独测试。  
Codex 负责辅助写代码和整理文件，Godot 负责运行、摆场景和验证结果。
